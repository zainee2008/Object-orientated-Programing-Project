# Object Oriented Programming Project

This project is a Java application developed for QUB’s Photography Department. It manages and organises a collection of photographic images across different genres.

## Features
- Add new images
- Search for images
- Display all stored images
- Organise images into albums

## Technologies Used
- Java
- Object-Oriented Programming

## Project Description
This application was designed to store image details, allow users to search and display images, and support the creation of image albums for display purposes. The system is menu-driven and provides a simple way to manage photographic records.

## How to Run
1. Open the project in a Java IDE such as Eclipse or IntelliJ.
2. Make sure all source files are in the correct folders.
3. Run the main class file.
4. Use the menu options to add, search, and display images.
